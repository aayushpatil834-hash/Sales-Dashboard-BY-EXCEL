# 📊 Superstore Sales Dashboard (Excel)

## Overview

This project is an interactive Sales Dashboard built in Microsoft Excel to analyze sales performance, customer insights, and profit trends. It uses Pivot Tables, Pivot Charts, Slicers, and Maps to provide dynamic business insights.

## Features

- Interactive Year & Category filters
- Sales by Category
- Profit Trend Analysis
- State-wise Sales Map
- Customer Count
- Monthly Sales Analysis
- Top 5 Customers by Profit
